## Стратегія лінкбілдінгу

Рекомендований розподіл анкорів:

- 50% — голий URL / бренд (`t-mebel.com.ua`, `T-Mebel`)
- 35% — частковий збіг (`меблі харків`, `кухні на замовлення`)
- 15% — точний ключ (`меблі на замовлення харків`)

## Для дослідження конкурентів

При отриманні нових даних по конкурентах — аналізувати:

1. Зміни позицій відносно попереднього виміру
2. Приріст посилань у топ-5 (чи є різкі стрибки)
3. Нові конкуренти що з'явились в топ-10
4. Контентна стратегія лідерів (які сторінки/статті у них ранжуються)
